InBase

interface InBase

Represents the numeric base component used in a unit prefix.

Prefix multipliers are typically calculated as base^power, where base is defined by this interface and power is the exponent associate with the specific prefix (e.g. 3 for kilo, -2 for centi).

This interface allows for different base systems to be used depending on the context, such as binary (base 2), decadic (base 10), or decimal (base 1000, often used for storage capacities).
