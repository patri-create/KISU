Scalar

class Scalar<A : Prefix<A>, System<A>>(prefix: A, unit: Unit) : Expression<Scalar<A>> , System<Scalar<A>>

Represents a scalar unit, composed of a Prefix and a unit symbol (e.g., "km" for kilo + meter, "μs" for micro + second).

This is the atomic element of the Expression system, which models complex unit compositions such as "N⋅m" or "mol⋅K".

A Scalar combines a prefix and a base unit into a single symbolic component, and can be composed with other expressions through multiplication and division to form Product and Quotient types.

It delegates System behavior to a ScalarSystem built from the same prefix and unit.
