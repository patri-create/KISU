Unit

class Unit(symbol: String, exponent: Exponent = Exponent(1)) : Comparable<Unit>

Represents a physical unit (such as "m", "s", "kg") with an optional exponent.

This class is used to model unit symbols and their powers, and supports basic arithmetic operations (multiplication and division) with units of the same type.

Equality and hash code are based only on the symbol name, not on the exponent. This is useful when you want to treat units like "m²" and "m³" as the same base unit for lookup or comparison purposes.
